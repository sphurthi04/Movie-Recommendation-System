# 🎬 Movie Recommendation System

A content-based Movie Recommendation System that suggests similar movies based on user input using cosine similarity. 

---

## 📌 Project Overview

This project recommends movies similar to a given movie by analyzing content features such as genres, keywords, cast, and crew. It uses a similarity matrix to find and display the most relevant movies.

---

## 🚀 Features

* 🔍 Search movies by name
* 🎯 Recommends top 5 similar movies
* ⚡ Fast and efficient recommendation system
* 🧠 Uses cosine similarity for accurate results

---

## 🛠️ Technologies Used

## ⚙️ How It Works

1. Data preprocessing and feature extraction
2. Conversion of movie data into vectors
3. Calculation of similarity using cosine similarity
4. Recommending top 5 similar movies

## ▶️ How to Run

1. Clone the repository:

```
git clone https://github.com/sphurthi04/Movie-Recommendation-System.git
```

2. Navigate to the project folder:

```
cd Movie-Recommendation-System
```

3. Run the application:

```
python app.py
```

4. Enter a movie name and get recommendations
